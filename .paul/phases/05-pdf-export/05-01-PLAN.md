<objective>
## Goal
Add PDF report generation with Stardew Valley-styled layout, server-side charts, and a download button in the dashboard. Add bonus statistics and capitalist comedy polish throughout.

## Purpose
Players want to share farm reports with friends or keep records outside the game. PDF export turns the dashboard data into a shareable, printable document with the same pixel art personality.

## Output
- PdfReportService: generates styled PDF reports per season or full history
- ScottPlot charts rendered server-side as PNG for PDF embedding
- /api/report/pdf endpoint to download the report
- "Export PDF" button in dashboard header
- Bonus stats: ROI per crop, best/worst seasons
- Comedy Easter eggs in PDF (Valerie quotes, URSSAF disclaimers)
- Updated documentation
</objective>

  <name>Task 1: Add NuGet packages and create PdfReportService</name>
  <files>LaCompta/LaCompta.csproj, LaCompta/Services/PdfReportService.cs</files>
  <action>
    1. Add NuGet packages:
       - QuestPDF (2024.3.x — community license, free for open-source)
       - ScottPlot (5.x — MIT license, .NET chart rendering)

    2. Create LaCompta/Services/PdfReportService.cs:
       - Constructor takes Repository, IMonitor
       - Method: GenerateReportAsync(string playerId) → byte[] (PDF bytes)
       - Method: GenerateReportToFileAsync(string playerId, string outputPath)
       - Internal: BuildTitlePage — farm name, player, date, comedy tagline
       - Internal: BuildSeasonPage(season, year) — table with daily data + ScottPlot bar chart
       - Internal: BuildSummaryPage — all-time stats, best/worst season, total ROI
       - Internal: RenderCategoryBarChart(season) → byte[] PNG via ScottPlot
       - Use QuestPDF fluent API: Document.Create → Page → Column → Table
       - Set QuestPDF.Settings.License = LicenseType.Community in static constructor
       - Color scheme matching dashboard: dark background text on white PDF paper
       - Category colors: Farming=#4CAF50, Foraging=#8BC34A, Fishing=#2196F3, Mining=#FF9800, Other=#9E9E9E
       - Comedy elements: URSSAF disclaimer footer, Valerie quotes between sections
       - Bonus stats: best season (highest income), worst season, top item by profit, total ROI

    Avoid:
    - Chart.js (browser-only) — use ScottPlot for server-side rendering
    - Blocking game thread — generate on background thread
    - Huge PDF files — limit chart resolution, compress images
  </action>

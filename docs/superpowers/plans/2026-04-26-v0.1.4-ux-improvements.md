# v0.1.4 UX Improvements Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.1.4 with auto-refresh of the dashboard on day change (item #1) and a one-time dismissable mid-history banner (item #4) on top of the auto-open default flip already committed (item #3).

**Architecture:** Frontend-only polling on the existing `/api/farminfo` endpoint detects day changes and triggers a soft re-render via the existing `loadPageData(page)` dispatcher. A new key/value `mod_state` table in each save's DB persists the banner-dismissed flag; two new endpoints (`GET /api/state`, `POST /api/state/dismiss-mid-history`) read/write it. The banner appears when the chronologically-earliest `daily_records` row is not Spring Y1 D1 (or, when the table is empty, when the current in-game day is not Spring Y1 D1).

**Tech Stack:** C# .NET 6, SMAPI 4.x, `Microsoft.Data.Sqlite`, vanilla JS in `Web/Assets/app.js`, ModBuildConfig.

**Spec reference:** `docs/superpowers/specs/2026-04-26-v0.1.4-ux-improvements-design.md`

**Test infrastructure note:** No automated test runner. Verification is `dotnet build` + final manual smoke-test in Stardew, per the spec's testing protocol.

**Item #3 was already implemented** in commit `089766b` on `main` (the `AutoOpenDashboard` default flip). It's part of the v0.1.4 release notes but not part of this plan.

        public string? GetModState(string key)
        {
            using var conn = _db.GetConnection();
            var cmd = conn.CreateCommand();
            cmd.CommandText = "SELECT value FROM mod_state WHERE key = $key";
            cmd.Parameters.AddWithValue("$key", key);
            var result = cmd.ExecuteScalar();
            return result == null || result == System.DBNull.Value ? null : (string)result;

        public void SetModState(string key, string value)
        {
            using var conn = _db.GetConnection();
            var cmd = conn.CreateCommand();
            cmd.CommandText = "INSERT OR REPLACE INTO mod_state (key, value) VALUES ($key, $value)";
            cmd.Parameters.AddWithValue("$key", key);
            cmd.Parameters.AddWithValue("$value", value);
            cmd.ExecuteNonQuery();
        }

        // === First-record-day (chronologically earliest daily_records row) ===

        /// <summary>
        /// Returns the (year, season, day) of the chronologically earliest daily_records row,
        /// or null if no records exist yet. Season ordering follows Stardew's calendar
        /// (spring &lt; summer &lt; fall &lt; winter), not lexicographic order.
        /// </summary>
        public (int Year, string Season, int Day)? GetFirstRecordDay()
        {
            using var conn = _db.GetConnection();
            var cmd = conn.CreateCommand();
            cmd.CommandText = @"
                SELECT season, year, day FROM daily_records
                ORDER BY year ASC,
                         CASE season
                             WHEN 'spring' THEN 1
                             WHEN 'summer' THEN 2
                             WHEN 'fall' THEN 3
                             WHEN 'winter' THEN 4
                         END,
                         day ASC
                LIMIT 1";
            using var reader = cmd.ExecuteReader();
            if (!reader.Read())
                return null;
            return (
                reader.GetInt32(reader.GetOrdinal("year")),
                reader.GetString(reader.GetOrdinal("season")),
                reader.GetInt32(reader.GetOrdinal("day"))
            );
        }

        // === Mappers ===

        private static DailyRecord MapDailyRecord(SqliteDataReader reader)

var lastSeenDay = null;
var pollIntervalId = null;
var POLL_INTERVAL_MS = 10000;

async function refreshGameData() {
  var data = await Promise.all([
    fetchJson(apiUrl('/api/summary')),
    fetchJson(apiUrl('/api/seasons'))
  ]);
  if (data[0]) summaryData = data[0];
  if (data[1]) seasons = data[1];
  var hash = window.location.hash || '#overview';
  var page = hash.replace('#', '') || 'overview';
  loadPageData(page);
}

function startAutoRefreshPolling() {
  if (pollIntervalId !== null) return;
  pollIntervalId = setInterval(async function() {
    var info = await fetchJson('/api/farminfo');
    if (!info || !info.isWorldReady) return;
    var key = info.year + '/' + info.season + '/' + info.day;
    if (key === lastSeenDay) return;
    lastSeenDay = key;
    await refreshGameData();
  }, POLL_INTERVAL_MS);
}
